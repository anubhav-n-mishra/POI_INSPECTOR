# Demo Script for POI Blueprint Quality Inspector

## 🎬 1-2 Minute Demo Flow

### Opening (10 seconds)
**Show landing page**

"Hi! This is the POI Blueprint Quality Inspector - an AI-powered system that verifies Point-of-Interest polygon accuracy for advertising platforms like GroundTruth."

### Problem Statement (15 seconds)
**Highlight features section**

"POI polygons often have errors - wrong shapes, boundaries cutting roads, overlapping stores. Even small errors cause wrong visit detection and bad campaign reporting."

### Solution Overview (15 seconds)
**Point to 'How It Works' section**

"Our system automatically:
1. Fetches satellite imagery
2. Detects building footprints using AI
3. Calculates quality metrics
4. Provides a 0-100 score with actionable suggestions"

### Live Demo (45 seconds)

**Click "Try Demo Analysis"**

"Let me show you a live analysis..."

**Analysis view appears**

"Here's our analysis dashboard. We get:

- **Overall Quality Score**: 78.5 out of 100, Grade B - Good quality
- **Key Metrics**:
  - IOU (alignment): 72% - decent overlap
  - Leakage: 18.5% - some area outside building
  - Coverage: 85% - most of building is covered
  - Regularity: 68 - reasonably rectangular shape

**Point to suggestions panel**

"And here are smart suggestions:
- 'Moderate leakage - shrink east side'
- 'Consider adjusting boundaries'

These are actionable corrections that improve polygon accuracy."

### Technical Highlights (15 seconds)

**Scroll to show UI**

"The system uses:
- Pretrained computer vision for building detection
- OpenStreetMap satellite tiles
- Weighted scoring algorithm
- Modern React dashboard with real-time analysis"

### Business Impact (10 seconds)

"This directly addresses GroundTruth's Blueprint technology needs - ensuring high-precision POI boundaries for accurate visit detection and campaign measurement."

### Closing (5 seconds)

**Show download button**

"Results can be exported as PDF reports. Thank you!"
